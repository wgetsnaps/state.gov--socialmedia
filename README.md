# Bash and wget commands

To create this mirror:

~~~sh
wget --page-requisites \
     --no-directories \
     --adjust-extension \
     --convert-links \
     --backup-converted \
     http://www.state.gov/r/pa/ode/socialmedia/ |
     tee ./wget.log
~~~
